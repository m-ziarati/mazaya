By default, the control will remember the last tab clicked. That tab will be in display when the page is reloaded or revisited.

Note: If you open this page directly with physical path(i.e. file:///D:/tabcontent/demo.html ) instead of from web app, you may not see this feature.

Setting <ul class="tabs" persist="false">...</ul> will turn off this feature.

You can specify the default tab to be shown by <li class="selected">...</li>.
You can have multiple tab contents on the same page, with only one copy of the CSS and JavaScript files.
You can also open a tab from a link anywhere on the page.
